# 🏠 Home Lab

A personal home lab powered by an old laptop, built for learning, experimentation, and self-hosting modern technologies. This repository documents the setup, configurations, services, and projects running in my home lab environment.

## 🚀 Overview

This home lab serves as a playground for exploring various technologies, including:

* Linux Administration
* Docker & Containerization
* Kubernetes
* Virtualization
* Self-Hosted Applications
* Networking & Security
* Infrastructure as Code (IaC)
* Monitoring & Observability
* CI/CD Pipelines
* Automation & Scripting
* Cloud-Native Technologies

The goal is to gain hands-on experience with production-like environments using affordable and repurposed hardware.
